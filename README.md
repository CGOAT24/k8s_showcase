This project demonstrates a minimal Kubernetes-based microservices architecture using:

- Kubernetes
- Dotnet Core
- Go
- MongoDb

## Installation

Requires: [Docker](https://docs.docker.com/), [Kubectl](https://kubernetes.io/releases/download/), [Minikube](https://minikube.sigs.k8s.io/docs/) and [Just](https://github.com/casey/just)
```
just go_build
just dotnet_build
just go_deploy
just dotnet_deploy
```
